This plugin brings a gutenberg slider block to the new WordPress Gutenberg editor.

== Description ==

Candy Slider is the collection of slider blocks for the new Gutenberg WordPress block editor.

= See Demo =
https://www.youtube.com/watch?v=pOC5qX5_Hfk

= COMPANION THEME =
We also created the [DO theme](https://themantiko.com?utm_medium=wp.org&utm_source=wordpressorg&utm_campaign=readme&utm_content=candyslider) in the Themantiko theme shop as the perfect companion for Candy Slider.

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Candy Slider"
3. **Install** the "Candy Slider" plugin
4. **Activate** "Candy Slider" from your Plugins page
5. **Insert Block** on your Gutenberg Editor and select "Candy Slider"
